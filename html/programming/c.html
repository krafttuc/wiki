1.3 The for statement

基本格式： 

int i;
for (i = 0; i < 10; i++){
	do something;
	...
}

顺序：

  1. 初始化i；
  2. 条件测试，若为真，则执行3，反之则执行5；
  3. 执行大括号内语句；
  4. 改变i，如此处为i++,执行2；
  5. 跳出；

1.5.1 File Copying

使用 putchar() 和 getchar() 实现最简单的copy功能。 

#include <stdio.h>
main()
{
	int c;
	c = getchar();
	while (c != EOF){
		putchar(c);
		c = getchar();
	}
}

or 

#include <stdio.h>
main()
{
	int c;
	while ((c = getchar()) != EOF){
		putchar(c);
	}
}

为什么不使用 char 类型来定义c？ 因为需要c足够大来容纳EOF和任何getchar返回的值

About EOF

EOF is an integer defined in <stdio.h>, but the specific numeric value doesn't matter as long as it is not the same as any char value. By using the symbolic constant, we are assured that nothing in the program depends on the specific numeric value.

实际上EOF = -1
